✓ data.json downloaded! How to publish the updated table:
1. Open your GitHub repository
2. Click Add file → Upload files (or drag the file into the window)
3. Select the downloaded data.json – it replaces the old one automatically
4. Click "Commit changes" at the bottom
5. After 1–2 minutes everyone sees the updated table! 🎉
You never need to touch index.html again.
